🎯 Key Academic Insights Applied

• CoVe's "independent answering" — don't re-read your own text before verifying
• FIRE's iterative retrieval — search → insufficient → search again until convergence
• IFCN's ≥2 source standard — every ⚠️ claim needs at least 2 unrelated sources
• Time-sensitive forced external verification — hours, prices, schedules change; even "yesterday's source" can be wrong
• Multi-agent anti-pattern — domain-split agents (housing/route/food) waste overlap. Correct: structural rounds (source audit → verification → impact)

🧬 Hybrid Design — Best of Both Worlds

From Journalism: Round 1 (SIFT — source attribution first), Round 4 (Truth Sandwich — structured correction)
From Academia: Round 2 (CoVe independent verification + FIRE iterative search), parallel multi-agent verification
Adversarial adaptation mitigated by ≥2 independent sources. Verification imperfection absorbed by FABLE impact analysis — not every error is worth fixing. Cost bounded by trigger conditions (≥300 chars, ≥5 claims). Motivated reasoning — the ceiling no framework has solved.
Lessons From Production

Hard-Won Pitfalls

Errors that actually happened. Each one taught us a new rule.

  • 🐙 Don't Fabricate Nutrition Data
    "Octopus has 15g protein" ← I made this up. Actual: 29.8g (USDA).
    A "reasonable-sounding" guess is almost always wrong.
  • ⚠️ Dual-Source Contamination
    Tour brochure ≠ user's actual booking. UMSL group stayed at VILLAJARDÍN, user booked Pons Minea.
    Round 1 must tag *intended* source (user vs document), not just *literal* source.
  • 💰 1688 Wholesale ≠ Retail Price
    Same item: 1688 ¥15-30 / Taobao ¥50-80 / Amazon $23-60. Always check ≥3 platforms.
    Secondhand/jump prices cannot be automated — mark as "estimate".
  • 📊 GPU Benchmarks: Don't Mix Sources
    GFXBench Car Chase: N2=104fps (onscreen vs offscreen confusion)
    Community scores vary wildly by thermals, firmware, ambient temp. Stick to GSMArena review page 4 or Notebookcheck UL Solutions.
  • ⚡ OPPO Pen Compatibility
    OPPO Pen uses Goodix GP850 active capacitive protocol — no EMR digitizer layer. Only compatible with Find N2/N3/N5/OnePlus Open (opposhop.cn verified).
  • 🕐 Published "Yesterday" May Still Be Wrong
    Time-sensitive items (hours, prices, schedules) always need current verification. Even a source published 24 hours ago can be outdated.
Logic & Theory Verification

Facts Are True. But Is The Argument Valid?

Facts may be correct but the reasoning connecting them can still be fallacious. Logic-checking catches what fact-checking misses: invalid causal inferences, circular arguments, false dichotomies, and statistical fallacies.

🧬 Example: When Facts Are True But Logic Is Flawed

Fact: "CO₂ is a greenhouse gas" → ✅ True
Fact: "Global temperatures increased 1.1°C" → ✅ True
Claim: "Therefore CO₂ increase after temperature means CO₂ doesn't cause warming" → ❌ Post hoc ergo propter hoc
Fact-check confirms both facts. Logic-check catches the invalid causal inference.
