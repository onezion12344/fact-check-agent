LLM Fallacy Detection by Type

LLMs show strong variance across fallacy categories (from Jeong et al. 2025, NAACL).

✅ Strong

Circular reasoning, appeal to popularity, slogans, thought-terminating clichés

Simple structure, fixed patterns

🟡 Medium

Straw man, false dichotomy, slippery slope, appeal to emotion, red herring

Requires reasoning decoding, confidence fluctuates

❌ Weak

False analogy, burden of proof evasion, statistical fallacies, Simpson's paradox

Needs background knowledge + multi-step reasoning

Facts vs Logic

Fact-checking verifies individual claims. Logic-checking verifies how those claims are connected. Both are needed.

DimensionFact-Check AgentLogic & Theory Check
TargetIndividual factual claimsArgument structure + reasoning
MethodSIFT → CoVe+FIRE → FABLE → Truth SandwichToulmin → FLICC → Prompt Ranking → FoVer

Output✅ Correct / ⚡ Disputed / ❓ Not found / ❌ Wrong✅ Valid / ⚡ Questionable structure / ❌ Fallacious
When usedPrice/time/address/stat checksPolitical/scientific/ethical arguments

🧬 Example: When Facts Are True But Logic Is Flawed

Fact: "CO₂ is a greenhouse gas" → ✅ True
Fact: "Global temperatures increased 1.1°C" → ✅ True
Claim: "Therefore CO₂ increase after temperature rise means CO₂ doesn't cause warming" → ❌ Flawed causal reasoning (post hoc ergo propter hoc)
Fact-check would confirm both facts. Logic-check catches the invalid causal inference.
