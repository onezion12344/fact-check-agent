4 Architecture in Action

🔵 Hermes Plugin (orchestrator)

pre_llm_call → detect factual intent → inject verification instruction. ~100 lines.

🟢 Main Agent (worker)

Reads injected context → calls delegate_task → spawns sub-agent for deep search.

🟡 Sub-agent (verifier)

Full tool access (web_search, browser) → executes CoVe+FIRE → returns structured result.
