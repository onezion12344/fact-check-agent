# 学术论文引用事实核查 — 子模式

## 适用场景

当核查对象是学术研究综述/元分析报告，包含大量论文引用（5+篇论文，20+条事实主张）时。

## 触发条件

- 用户要求对"研究报告"进行事实核查
- 报告包含：论文标题、作者、年份、期刊名、引用数、样本量、效应量、统计值（BF值/p值/相关系数）
- 用户说"事实核查"后补充"网站呈现"

## 与标准 Double-Check 流程的差异

| 维度 | 标准流程 | 学术核查模式 |
|:-----|:---------|:------------|
| **第1轮SIFT源** | 用户提供的文档/截图/网页 | 从我之前**提取的论文全文**（PMC/Springer全文） |
| **SIFT方法** | 逐句对照用户源 | 逐条主张对照中英文原文关键句（不paraphrase） |
| **CoVe目标** | 价格/地址/营业时间 | 引用数/DOI/期刊名/效应量/统计值 |
| **外验工具** | Google Maps / 电商 | 学术搜索引擎（AnySearch academic + PMC + DOI解析） |
| **输出格式** | 默认内联浓缩 | 用户常要求HTML（"网站呈现"） |

## 第1轮 SIFT 特殊要点

### 优先使用已提取的论文全文

本session中成功使用了已提取的全文：
- Mullin 2013 — PMC全文~2000行
- Chen 2018 — PMC全文~1500行
- Henry 2025 — Springer全文~800行
- Kelley 2019 — PMC全文~3000行
- Kroese 2014 — Frontiers+PMC双源全文
- Magalhães 2020 — PMC全文

**方法：** 不重新搜索，直接在上下文中的提取物里搜索关键词来核对每个主张。

### 可逐字核验的主张类型

从论文提取物中可以直接核验：
- **样本量**：如 "27 participants" / "808 Chinese college students"
- **统计值**：如 "BF10 = 1.17 × 10⁸" / "r = 0.158**" / "β = -0.52"
- **研究设计**：如 "within-subjects crossover design" / "EMA 4-week"
- **核心结论**：如 "opponent process dynamics in high-incongruence participants"
- **效应量描述**：如 "SSS → OSAs → PIPU mediated" / "57.5% variance in men"

### 需要标记⚠️的主张类型

- **精确效应量数值**（如 d=0.5-1.0）— 需要全文验证
- **论文声称的原创发现 vs 通用模型**（如Love 2015引用的"脱敏-敏化"是Robinson & Berridge 1993的模型，不是该论文的原创）
- **跨研究推断**（如"疲劳时想要>喜欢"来自Kelley 2019和Robinson & Berridge 2024的综合，非单篇论文）

## 第2轮 CoVe 特殊要点

### 需要外验的学术细节

对学术引用发现⚠️后，外验方法：

| 需要验证的项 | 验证方法 |
|:------------|:---------|
| 论文是否存在 | 搜索 DOI / 标题 + 第一作者 |
| 引用数 | 搜索后看搜索结果中的引用数标注 |
| 期刊名 | 搜索结果中提取期刊信息 |
| 研究类型 | 查看摘要确认是RCT/综述/元分析/原创研究 |
| 具体效应量 | 尝试提取全文；付费墙时改为定性描述 |

### 外验结果标注

| 状态 | 含义 |
|:----|:-----|
| ✅ 确认 | DOI+期刊+引用数均匹配，或有全文可对照 |
| ❌ 错误 | 作者/年份/期刊/数值与来源冲突 |
| ⚠️ 精度调整 | 论文存在结论正确，但需要修正表述方式（如标注"综述汇总"而非"原创发现"） |

### 付费墙处理

当论文因付费墙（DOI 403 / PDF不可提取）无法直接验证时：
- 引用数可以通过搜索结果交叉确认 ✅
- 具体效应量**不引用未验证的精确数值**，改为定性描述（"经元分析确认效果显著"而非"d=0.5-1.0"）

## 第3轮 FABLE 特殊要点

学术引用核查的FABLE分级：

| 级别 | 含义 | 示例 |
|:----|:-----|:-----|
| 🟢 Cosmetic | 引用方式需微调，不影响结论 | "该综述提出了模型"→"该综述汇总了模型" |
| 🟡 Experience | 效应量数值有偏差但方向正确 | d值差0.2但仍在同一量级 |
| 🔴 Critical | 引用完全错误，核心主张站不住 | 把case report说成RCT；把反方向结论说成正相关 |

## 第4轮 Truth Sandwich 特殊要点

学术修正的输出格式：

```
### 修正: [引用标签]

**原话：** 我写的版本

**更正：** 修正后的版本

**原因：** 具体说明为什么需要修正（如：该模型起源是Robinson & Berridge 1993，Love 2015是应用综述）

**来源：** 论文DOI / 页码 / 原文句子
```

## 完整流水示例（本session）

```
Step 1: 用户说"事实核查网站呈现cf给我"
Step 2: 判断为学术核查模式（报告含14篇论文引用）
Step 3: SIFT使用已有提取物核验可验证项
  → Mullin 2013: ✅ 全部逐字确认
  → Chen 2018: ✅ 全部逐字确认
  → Henry 2025: ✅ 全部逐字确认（含BF10精确值）
  → Burch&Salmon 2022: ⚠️ PDF不可提取
  → Love 2015: ⚠️ 403付费墙
Step 4: CoVe外验⚠️项
  → Burch&Salmon: ✅ 通过EvoS官网摘要+PsyPost双源确认
  → Love 2015: ⚠️ 引用数✅ 但模型归属需修正
Step 5: FABLE分级 → 🟢 2处Cosmetic
Step 6: Truth Sandwich + HTML输出
Step 7: 用户要求加Notion → 归档
```
